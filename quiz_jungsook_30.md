# 나는 SOLO 30기 정숙 스타일 적합도 퀴즈

작성일: 2026-03-23  
기준: 공개 기사/공식 클립에서 드러난 `나는 SOLO 30기 정숙`의 방송 페르소나  
목적: 사용자가 `30기 정숙 스타일`과 얼마나 잘 맞는지 엔터테인먼트형으로 판단하는 퀴즈 설계

---

## 1. 포지셔닝 문구

이 퀴즈는 `실제 정숙本人과의 궁합`을 측정하는 테스트가 아닙니다.  
공개 방송에서 드러난 `30기 정숙의 말투, 대화 태도, 관계 텐션, 표현 스타일`을 기준으로 한 **스타일 적합도 테스트**입니다.

권장 결과 설명 문구:

- `방송 속 30기 정숙 스타일과의 대화/관계 적합도입니다`
- `실제 성격, 실제 연애 성사 가능성, 실제 궁합을 보장하지 않습니다`

---

## 2. 측정 축

이 퀴즈는 아래 5개 축을 측정합니다.

### 1. 진정성

- 과한 이미지 관리보다 자연스러움을 선호하는지

### 2. 솔직함

- 불편함이나 감정을 지나치게 포장하지 않고 표현하는지

### 3. 주도성

- 관계에서 필요할 때 먼저 움직일 수 있는지

### 4. 배려형 실용성

- 상대를 편하게 해주면서도 상황을 실용적으로 정리하는지

### 5. 담백한 텐션

- 과장된 연출보다 편안하고 안정적인 분위기를 만드는지

---

## 3. 답변 방식

각 문항은 1점부터 5점까지 선택합니다.

- 1점: 전혀 아니다
- 2점: 별로 아니다
- 3점: 보통이다
- 4점: 어느 정도 그렇다
- 5점: 매우 그렇다

## 5. 점수 로직

## 5-1. 문항별 점수 변환

정방향 문항:

- 선택 점수를 그대로 사용

역방향 문항:

- 변환 점수 = `6 - 선택 점수`

예시:

- 역방향 문항에서 5점을 선택하면 실제 반영 점수는 1점
- 역방향 문항에서 2점을 선택하면 실제 반영 점수는 4점

## 5-3. 축별 점수 계산

각 축은 3개 문항으로 구성됩니다.

축 점수 계산식:

```text
축 점수 = ((문항합 - 3) / 12) * 100
```

설명:

- 최소 합계는 3점
- 최대 합계는 15점
- 이를 0~100점으로 변환

예시:

- 진정성 문항 합이 12점이면
- 진정성 점수 = `((12 - 3) / 12) * 100 = 75점`

---

## 5-4. 최종 적합도 계산

최종 적합도는 가중합으로 계산합니다.

```text
최종 적합도 =
(진정성 * 0.30) +
(솔직함 * 0.20) +
(주도성 * 0.20) +
(배려형 실용성 * 0.15) +
(담백한 텐션 * 0.15)
```

가중치 이유:

- 공개 기사상 `정숙 스타일`에서 가장 강하게 반복된 축은 `진정성`과 `이미지 과잉에 대한 피로감`
- 다음으로 `솔직함`, `주도성`
- 그 다음이 `배려형 실용성`, `담백한 텐션`

## 5-5. 결과 등급

### 85~100점

`매우 잘 맞음`

설명 예시:

- 방송 속 30기 정숙 스타일과 상당히 잘 맞는 편입니다
- 자연스러움, 솔직함, 관계 텐션에서 높은 적합도가 보입니다

### 70~84점

`잘 맞는 편`

설명 예시:

- 전반적으로 30기 정숙 스타일과 잘 맞는 편입니다
- 특히 대화 방식이나 관계 접근법에서 안정적인 호환성이 보입니다

### 55~69점

`맞는 부분과 다른 부분이 공존`

설명 예시:

- 맞는 포인트도 분명하지만 몇몇 관계 텐션에서는 차이가 있을 수 있습니다
- 어떤 축에서 점수가 갈렸는지 함께 보여주는 것이 좋습니다

### 40~54점

`조금 다름`

설명 예시:

- 30기 정숙 스타일과는 관계 템포나 표현 방식에서 차이가 있을 수 있습니다
- 나쁜 결과가 아니라 선호하는 대화 방식이 다른 쪽에 가깝다는 의미입니다

### 0~39점

`스타일 차이 큼`

설명 예시:

- 방송 속 30기 정숙 스타일과는 매력 포인트가 꽤 다를 수 있습니다
- 특히 진정성, 솔직함, 관계 주도성 축에서 차이가 클 가능성이 있습니다

## 6. 축별 해석 문구

최종 점수만 보여주기보다 축별 해석을 같이 보여주는 것이 좋습니다.

## 진정성

- 80점 이상: `꾸밈없는 태도와 자연스러운 관계를 선호합니다`
- 50~79점: `자연스러움을 중요하게 여기지만 상황에 따라 이미지 관리도 고려합니다`
- 49점 이하: `관계에서 인상 관리와 분위기 조절을 더 중시하는 편입니다`

## 솔직함

- 80점 이상: `불편함이나 감정을 비교적 분명하게 표현하는 편입니다`
- 50~79점: `필요할 때는 솔직하지만 관계 분위기도 함께 고려합니다`
- 49점 이하: `직설적 표현보다는 완곡한 소통을 선호합니다`

## 주도성

- 80점 이상: `마음이 가면 스스로 움직일 수 있는 타입입니다`
- 50~79점: `상황을 보며 움직이지만 필요할 때는 주도권을 잡을 수 있습니다`
- 49점 이하: `관계에서 먼저 움직이기보다 흐름을 보는 편입니다`

## 배려형 실용성

- 80점 이상: `상대를 편하게 해주면서도 상황 정리에 강한 편입니다`
- 50~79점: `배려와 현실 감각이 균형 잡힌 편입니다`
- 49점 이하: `상황 수습이나 감정 완충보다 자신의 반응이 먼저 나올 수 있습니다`

## 담백한 텐션

- 80점 이상: `과한 연출보다 편안하고 안정적인 대화 텐션을 선호합니다`
- 50~79점: `분위기와 편안함 사이의 균형을 보는 편입니다`
- 49점 이하: `정적인 텐션보다 활기 있고 표현적인 흐름을 선호합니다`

## 8. 서버 계산 예시

```ts
type Answers = Record<string, number>;

const reverseItems = new Set(["Q4", "Q6", "Q10", "Q12"]);

const dimensions = {
  authenticity: { weight: 0.30, items: ["Q1", "Q2", "Q11"] },
  directness: { weight: 0.20, items: ["Q3", "Q4", "Q12"] },
  initiative: { weight: 0.20, items: ["Q5", "Q6", "Q13"] },
  practicalCare: { weight: 0.15, items: ["Q7", "Q8", "Q14"] },
  calmTension: { weight: 0.15, items: ["Q9", "Q10", "Q15"] }
};

function normalizeScore(sum: number) {
  return ((sum - 3) / 12) * 100;
}

function getAdjustedScore(questionId: string, value: number) {
  return reverseItems.has(questionId) ? 6 - value : value;
}

function calculateResult(answers: Answers) {
  const dimensionScores: Record<string, number> = {};

  for (const [key, config] of Object.entries(dimensions)) {
    const sum = config.items.reduce((acc, itemId) => {
      return acc + getAdjustedScore(itemId, answers[itemId]);
    }, 0);

    dimensionScores[key] = Math.round(normalizeScore(sum));
  }

  const total = Math.round(
    Object.entries(dimensions).reduce((acc, [key, config]) => {
      return acc + dimensionScores[key] * config.weight;
    }, 0)
  );

  return {
    total,
    dimensionScores
  };
}
```

## 9. 운영 팁

- 처음 버전은 문항 15개면 충분합니다
- 결과 페이지에는 `총점 + 축별 레이더 차트 + 설명 문구`를 같이 보여주는 편이 좋습니다
- 가능하면 `왜 이런 결과가 나왔는지`를 2~3문장으로 풀어주는 요약도 넣는 것이 좋습니다
- 이 퀴즈는 특정 방송 출연자의 `공개 페르소나` 기반이므로, `실제 본인 성격/실제 궁합`처럼 보이게 만들지 않는 것이 중요합니다

## 10. 추천 결과 요약 템플릿

### 고적합 템플릿

당신은 30기 정숙 스타일과 꽤 잘 맞는 편입니다.  
특히 `진정성`, `솔직함`, `관계 텐션`에서 높은 점수를 보였습니다.  
과한 연출보다 자연스럽고 편안한 관계를 선호하는 점이 강점으로 보입니다.

### 중간 템플릿

당신은 30기 정숙 스타일과 맞는 부분도 있지만 차이가 나는 부분도 있습니다.  
대화 방식이나 관계 속도에서는 잘 맞을 수 있지만, 몇몇 표현 방식에서는 온도 차가 있을 수 있습니다.

### 저적합 템플릿

당신은 30기 정숙 스타일과는 조금 다른 매력 구조를 가진 편입니다.  
이 결과는 좋고 나쁨이 아니라, 선호하는 대화 톤과 관계 템포가 다를 수 있다는 의미입니다.
